Natural Language Processing

Duplicate Question Pair Detection

Developed a semantic similarity system to identify duplicate questions using the Quora Question Pairs dataset. I implemented a Siamese Manhattan LSTM (MaLSTM) architecture.

Unlike traditional cosine similarity, the MaLSTM uses the Manhattan distance (L1 norm) between the final hidden states of two weight-sharing LSTMs to compute semantic proximity.
